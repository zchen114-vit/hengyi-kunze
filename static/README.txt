This folder is for PWA (Progressive Web App) assets.

To make the app installable on phones:

1. Put real icon images here:
   - icon-192.png (192x192 pixels)
   - icon-512.png (512x512 pixels)
   - (Optional) icon-512-maskable.png

2. When self-hosting (not on Streamlit Cloud), you can link the manifest.

For Streamlit Community Cloud, full PWA support is limited.
The easiest way for users is still:
- Open in Safari/Chrome → "Add to Home Screen"

We will improve PWA support when we self-host or use Capacitor.
