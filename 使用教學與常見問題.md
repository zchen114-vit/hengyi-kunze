# 洞察易生的經歷 - 使用教學 + 常見問題（給使用者看）

## 快速開始

1. 打開連結
2. 側邊欄填寫你的稱呼 + 選擇偏好（傳統或現代）
3. 點擊想問的分區卡片
4. 輸入問題
5. 選擇解讀方式：
   - ⚡ 內建快速解讀（馬上給答案）
   - ✨ SuperGrok 專業解讀（推薦，品質更好）
6. 想換其他問題可以繼續問，或點「回到首頁」換分區

---

## 手機安裝成 App（PWA）

### iPhone（Safari）
1. 用 Safari 開啟連結
2. 點下方「分享」按鈕（方框+向上箭頭）
3. 往下滑找到「加入主畫面」
4. 按「加入」即可

### Android（Chrome）
1. 用 Chrome 開啟連結
2. 點右上角 ⋮ 三個點
3. 找到「安裝應用程式」或「新增至主畫面」
4. 確認安裝

安裝後桌面上會出現圖示，像一般 App 一樣開啟。

## 關於資料儲存（重要）

- 你的姓名、偏好、聊天記錄**只存在自己的手機/電腦**裡
- 關閉分頁或清除資料後，記錄可能會消失
- 開發者（我）**看不到**你問了什麼問題
- 使用 SuperGrok 解讀時，回答是在 grok.x.ai 產生的

**想長期保存**：請使用側邊欄的「匯出目前分區對話」功能下載存檔。

## 常見問題 FAQ

**Q1: 為什麼我的聊天記錄不見了？**
A: 這是網頁版設計，目前記錄只保存在瀏覽器。建議使用「匯出對話」功能保存重要內容。之後如果需要雲端保存可以再開發。

**Q2: SuperGrok 解讀要怎麼用？**
A: 
1. 輸入問題後點「使用 SuperGrok 專業解讀」
2. 複製下面出現的提示詞
3. 開新分頁去 https://grok.x.ai
4. 貼上問題並送出
5. 把回答複製回來貼到 App 裡，按「採用此回答」

**Q3: 可以跨手機和電腦同步嗎？**
A: 目前不行。每個裝置都是獨立記錄。

**Q4: 這是真正的易經起卦嗎？**
A: 不是。這是**直接用問題內容**請易經智慧給建議，沒有隨機搖卦過程。適合想直接問生活問題的人。

**Q5: 我想讓你親自幫我解讀，可以嗎？**
A: 可以！請在 Threads / IG 私訊我，或在 App 裡輸入問題後用「SuperGrok」模式把 prompt 發給我，我會親自回覆。

**Q6: 網址打不開怎麼辦？**
A: 可能是 App 在休眠，重新整理幾次，或等一下再試。

**Q7: 這要錢嗎？**
A: 完全免費，無限使用。
